# Elora Financeiro

O Elora é uma plataforma SaaS de gestão financeira pensada para organizar informações, acompanhar a saúde do negócio e apoiar decisões com clareza.

## Visão inicial

O projeto reunirá uma aplicação web, uma API e a infraestrutura de dados necessária para oferecer uma experiência segura, simples e escalável de gestão financeira.

## Estrutura do repositório

```text
.
├── assets/       # Identidade visual e outros recursos estáticos
├── backend/      # API, regras de negócio e integrações
├── docs/         # Documentação do produto e da arquitetura
├── frontend/     # Aplicação web
└── supabase/     # Banco de dados, autenticação e configurações Supabase
```
